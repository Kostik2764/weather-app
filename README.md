# WeatherApp

Погодное веб-приложение с регистрацией, аутентификацией и сохранением избранных городов. Позволяет искать погоду по названию города, просматривать текущие условия и прогноз на 5 дней, добавлять города в избранное. Администратор может управлять пользователями.

## Стек

- **Backend:** Node.js + Express
- **База данных:** SQLite (sql.js)
- **Аутентификация:** bcryptjs + express-session
- **Шаблонизатор:** EJS, Bootstrap 5
- **API погоды:** OpenWeatherMap (геокодирование + прогноз)
- **CI/CD:** GitHub Actions + Qlty (анализ качества)
- **Деплой:** Render

## Локальный запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Kostik2764/weather-app.git
   cd weather-app
Установите зависимости:

bash
npm ci
Создайте файл .env в корне проекта со следующим содержимым:

text
API_KEY=ваш_ключ_от_openweathermap
SESSION_SECRET=любая_строка_для_сессий
ADMIN_PASSWORD=пароль_для_администратора
Запустите сервер:

bash
npm start
Откройте в браузере: http://localhost:3000

Тестовые данные для входа
Роль	Логин	Пароль
Администратор	admin	значение ADMIN_PASSWORD из .env (сейчас по умолчанию логин admin пароль admin123)
Обычные пользователи регистрируются самостоятельно через форму регистрации.

Возможности
Регистрация и вход пользователей

Поиск погоды по названию города

Отображение текущей погоды и прогноза на 5 дней

Добавление городов в избранное

Удаление городов из избранного

Админ-панель:

просмотр всех пользователей

просмотр избранных городов любого пользователя

удаление пользователей
